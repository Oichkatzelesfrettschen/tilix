# Tilix User Guides

This directory will contain user and developer guides for Tilix.

## Planned Guides

### User Guides

- **Getting Started** - Installation and basic usage
- **Keyboard Shortcuts** - Complete keybinding reference
- **Profiles and Themes** - Configuration customization
- **Quake Mode** - Drop-down terminal setup
- **Tips and Tricks** - Advanced usage patterns

### Developer Guides

- **Building from Source** - Detailed build instructions
- **Contributing Code** - Development workflow
- **Testing** - Running and writing tests
- **Debugging** - Debug tools and techniques
